# NovaRetail+ Data Project
Análisis correlacional de factores de comportamiento de clientes de NovaRetail+ para identificar relaciones con el ingreso anual. Incluye visualización y coeficientes de correlación (Pearson, Spearman, punto-biserial y V de Cramér).

Correlational analysis of NovaRetail+ customer behavior factors to identify relationships with annual revenue. Includes visualizations and correlation coefficients (Pearson, Spearman, point-biserial, and Cramer’s V).
